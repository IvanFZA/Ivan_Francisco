# Manual de Usuario

## Introducción

Este documento describe el uso básico del proyecto.

## Requisitos

* Git instalado
* Cuenta de GitHub
* Acceso al repositorio

## Uso

1. Clonar el repositorio.
2. Crear una rama de trabajo.
3. Realizar modificaciones.
4. Enviar cambios al repositorio remoto.
5. Crear un Pull Request.
