# Proyecto_IvanFrancisco

Proyecto creado como parte de las prácticas de Git y GitHub.

## Objetivo

Aprender el uso de Git para el control de versiones y GitHub para el almacenamiento remoto de repositorios.

## Temas practicados

* Inicialización de repositorios
* Commits
* Ramas
* Repositorios remotos
* SSH
* Push y Pull
* Clonación de repositorios
